## 설치방법

- firebase가 설치되어 있어야 배포가 가능함.

```cmd
$ npm install -g firebase-tools
```

- 로그인은 chaewonkong@gmail.com

```cmd
$ firebase login
```

## 배포 방법

```cmd
$ firebase deploy
```
